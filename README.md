# NumberApi

### Features
- [x] Usuário envia um número para a Api: Numbers Api.
- [x] Recebe a curiosidade do Número digitado.
- [x] Mostra a curiosidade do número na tela.
- [x] Após 6 segundos ocultar o resultado.
